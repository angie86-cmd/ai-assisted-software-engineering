# AI-Assisted Software Engineering

A collection of practical experiments and case studies exploring how Generative AI can support modern software engineering activities.

This repository focuses on the intersection of:

- Prompt Engineering
- Requirements Engineering
- Software Architecture
- Code Generation
- Testing and Validation
- Documentation
- AI-Assisted Development

## Motivation

After more than 15 years working in software engineering, systems engineering and requirements engineering, I became interested in understanding how Generative AI can support engineering activities throughout the software lifecycle.

The goal of this repository is not to replace engineering expertise, but to explore how AI can enhance productivity, quality and decision-making when combined with sound engineering practices.
