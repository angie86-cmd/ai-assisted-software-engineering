# SMID Compass - System Description

## Purpose

SMID Compass is an AI-powered guidance platform developed by Save My Identity (SMID).

The platform supports people facing identity-document challenges by providing structured, accessible and responsible information.

## Target Users

- Stateless persons
- Refugees
- Migrants
- NGOs
- Volunteers
- Community organizations

## Multi-Agent Architecture

SMID Compass is based on a multi-agent architecture.

### Router Agent

Receives user requests and routes them to the appropriate specialized agent.

### Passport Guidance Agent

Provides information related to identity documents and passport guidance.

### Workshop Advisor Agent

Provides information about SMID workshops, training sessions and educational activities.

### Safety Agent

Checks whether responses follow safety, privacy and governance requirements.

## Knowledge Sources

- Passport Guide
- Workshop Guide
- SMID documentation

## Constraints

- The system shall not provide legal advice.
- The system shall respect user privacy.
- The system shall avoid collecting unnecessary personal data.
- The system shall provide transparent AI-generated guidance.
